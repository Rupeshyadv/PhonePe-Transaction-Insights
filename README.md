# 📊 PhonePe Analytics Dashboard

An interactive data analytics dashboard built using **Streamlit** to analyze and visualize PhonePe transaction data across India. This project transforms raw JSON data into meaningful insights on transaction trends, user engagement, device usage, and insurance adoption.

---

## 🚀 Features

* 📈 **Transaction Analysis**

  * Trend analysis across years and quarters
  * Category-wise transaction distribution

* 🌍 **Geographical Insights**

  * State-wise transaction comparison
  * Identification of high-performing regions

* 📱 **User & Device Analytics**

  * Device brand distribution
  * User engagement analysis (App Opens vs Registered Users)

* 🛡️ **Insurance Insights**

  * Growth trends in insurance transactions
  * State-wise insurance adoption

* 🎯 **Interactive Dashboard**

  * Dynamic filters (Year, Quarter)
  * KPI metrics for quick overview
  * Automated insights for business understanding

---

## 🧠 Key Highlights

* Converts raw nested JSON data into structured datasets
* Handles real-world data inconsistencies and missing values
* Implements data wrangling and transformation pipelines
* Provides business-level insights, not just visualizations

---

## 🛠️ Tech Stack

* **Frontend / Dashboard**: Streamlit
* **Data Processing**: Python (Pandas)
* **Visualization**: Plotly
* **Data Source**: PhonePe Pulse Dataset

## ⚙️ Installation & Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/phonepe-analytics.git
cd phonepe-analytics
```

---

### 2. Install Dependencies

```
pip install -r requirements.txt
```

---

### 3. Run the Application

```
streamlit run dashboard.py
```
